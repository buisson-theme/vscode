# Buisson

> *From the French — a thicket, a tangle of wild growth at the edge of a path.*

An inky botanical color scheme for VS Code. Warm paper backgrounds, botanical ink accents, WCAG-calibrated contrasts.

Inspired by [Everforest](https://github.com/sainnhe/everforest) and [Flexoki](https://github.com/kepano/flexoki).

## Installation

Install from the [VS Code Marketplace](https://marketplace.visualstudio.com/items?itemName=adrius.buisson-theme).

Or via the command palette: `Extensions: Install Extensions` → search "Buisson".

Then: `Preferences: Color Theme` → **Buisson Dark** or **Buisson Light**.

## Palette

| Name | Dark | Light | Role |
|------|------|-------|------|
| Hibiscus | `#d04550` | `#c02040` | Keywords, booleans, exceptions |
| Sage | `#6aaa44` | `#387008` | Functions, method calls |
| River Moss | `#2ea882` | `#096868` | Types, classes, interfaces |
| Slate Sky | `#4878ba` | `#1860a8` | Numbers, numeric constants |
| Thistle | `#b070d0` | `#6028a8` | Operators, decorators |
| Ochre | `#c87838` | `#a04810` | Strings, template literals |

## Accessibility

- Primary text: **11.55:1 AAA** (dark) / **15.15:1 AAA** (light)
- All 6 accents: WCAG AA or above
